### Escolher um Modelo de IA

Escolha seu modelo de IA preferido através do [alternador de modelo](/advanced/models#model-dropdown), e o Cursor Chat usará esse modelo para gerar respostas. Você pode alternar entre modelos pressionando `Ctrl/⌘ /`.

Por padrão, o Cursor Chat usa o `GPT-4` da OpenAI para seu modelo de IA (especificamente, o rótulo `gpt-4` aponta para nossa instância do GPT4-Turbo).

O modelo de IA que você escolher para o Chat será salvo para sessões futuras, então você não precisará mudá-lo toda vez que abrir o Cursor Chat.

### No Editor

O Cursor Chat também pode ser usado como uma aba do editor clicando no botão horizontal "Mais" e, em seguida, clicando no botão "Abrir Chat no Editor".
O Cursor Chat então se comportará como uma aba regular do editor. Pressionar `Ctrl/⌘ + L` abrirá a aba de chat e focará nela.

### Configurações

Você pode personalizar o Cursor Chat em `Configurações do Cursor` > `Recursos` > `Chat`.

Essas configurações incluem:

- Sempre buscar respostas na web.
  - Isso fará com que o modelo de IA navegue na web a cada consulta para obter as informações mais atualizadas.
- Adicionar animação de desvanecimento no chat.
  - Isso adiciona uma animação suave às mensagens de IA à medida que são geradas.
- Padrão para nenhum contexto.
  - Isso fará com que o modelo de IA use apenas a mensagem do usuário como contexto, e nenhum contexto adicional, como o arquivo atual, será incluído.
- Rolagem automática do chat.
  - Isso rolará automaticamente o chat à medida que a IA gera texto quando estiver no final do thread.
- Barra de rolagem estreita no painel de chat.
- Mostrar histórico de chat ao iniciar um novo chat.
