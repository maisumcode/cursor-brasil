### Chat de Base de Código Padrão

Por padrão, suas pastas são indexadas no servidor do Cursor. O Cursor Chat primeiro analisa seu código localmente, calculando consultas relevantes, e depois acessa a base de código indexada para fornecer respostas mais precisas e contextualizadas.

### Pesquisa de Embeddings
Com o código totalmente indexado, o Cursor Chat gera respostas altamente precisas e contextualizadas baseadas em sua base de código completa.

O Cursor analisa inteligentemente sua base de código indexada para identificar trechos relevantes ao seu contexto atual. Para direcionar especificamente a busca na base de código, basta incluir `@codebase` em seu prompt.

Para gerenciar a indexação da base de código:
1. Abra as configurações do Cursor com `⌘ + Shift + P`
2. Navegue até "Features"
3. Localize "Codebase Indexing"
4. Clique em "Show Settings"
5. Ative ou desative conforme sua necessidade
