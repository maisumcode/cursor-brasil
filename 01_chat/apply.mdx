O recurso `apply` permite que você integre rapidamente um bloco de código sugerido pelo chat para aplicar as alterações diretamente no seu código.

  <img src="/images/chat/apply.png" alt="Demonstração do recurso Apply" />
</Frame>

## Vantagem

A principal vantagem do recurso Apply é a capacidade de implementar alterações de código instantaneamente. Enquanto você trabalha no editor, basta interagir com o chat, descrever sua necessidade e, ao receber uma sugestão de código, aplicá-la diretamente no seu arquivo com um simples comando. Isso elimina a necessidade de copiar e colar manualmente, tornando o processo de desenvolvimento mais ágil e eficiente.

## Revisão e Confirmação de Alterações

Após aplicar um bloco de código, você tem várias opções para revisar e confirmar as alterações:

### Usando a Interface Visual
- Examine os diffs destacados no editor
- Utilize os botões "Aceitar" ✓ ou "Rejeitar" ✗ localizados no canto superior direito do bloco de código

### Usando Atalhos de Teclado
- **Aceitar**: `Ctrl/⌘ + Enter`
- **Rejeitar**: `Ctrl/⌘ + Backspace`
