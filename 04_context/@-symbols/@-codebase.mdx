### `@Codebase`

<Frame>
  <img src="/images/chat/@codebase.png" />
</Frame>

Através do `@Codebase`, o Cursor Chat segue estas etapas até encontrar as partes mais importantes do código para usar.

- Coleta: escaneando sua base de código em busca de arquivos importantes / trechos de código
- Reclassificação: reordenando os itens de contexto com base na relevância para a consulta
- Raciocínio: pensando em um plano de uso do contexto
- Geração: criando uma resposta

Outra maneira de enviar uma consulta avançada de base de código é clicar no menu suspenso ao lado do botão `Ctrl/⌘ + Enter` e selecionar `reranker` para o comportamento de busca.
Isso só está disponível quando `@Codebase` não é usado, caso contrário, `@Codebase` tem precedência.
