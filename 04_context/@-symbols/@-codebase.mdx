### `@Codebase`

<Frame>
  <img src="/images/chat/@codebase.png" />
</Frame>

O Cursor Chat segue algumas etapas para encontrar as partes mais importantes do código que vai usar.

- Coleta: escaneando sua base de código em busca de arquivos importantes.
- Reclassificação: reordenando os itens de contexto com base na relevância para a consulta.
- Raciocínio: pensando em um plano de uso do contexto
- Geração: criando uma resposta

Quando você usa `@Codebase` ele vai aparecer a opção para vc incluir ou excluir arquivos.
