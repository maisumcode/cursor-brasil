title: Uso Básico
---

Nas caixas de entrada de IA do Cursor, como no Cmd K, Chat ou Terminal Cmd K, você pode usar símbolos @ digitando `@`. Um menu pop-up aparecerá com uma lista de sugestões, e ele filtrará automaticamente para mostrar apenas as sugestões mais relevantes com base na sua entrada.

### Atalhos de Teclado

Você pode navegar pela lista de sugestões usando as teclas de seta para cima/baixo. Você pode pressionar `Enter` para selecionar uma sugestão. Se a sugestão for uma categoria, como `Arquivos`, as sugestões serão filtradas para mostrar apenas os itens mais relevantes dentro dessa categoria.

### Atalho de Teclado Cmd K

Você pode usar a tecla de seta para cima/baixo para navegar pela lista de símbolos @ selecionados no Cmd K, `Enter` para expandir/colapsar o item de contexto selecionado. Para referências de arquivos, você pode usar `Ctrl/⌘ M` para alternar as estratégias de leitura de arquivos. Leia mais sobre estratégias de leitura de arquivos [aqui](/context/@-symbols/@-files#cmd-k-chunking-strategy).
