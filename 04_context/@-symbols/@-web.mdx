### `@Web`

O recurso `@Web` permite que o Cursor construa uma consulta de busca com base na sua entrada e no contexto fornecido, pesquisando na web para encontrar informações relevantes. Isso é particularmente útil para encontrar as informações mais atualizadas ou para obter contexto adicional que não está disponível localmente.

## Como Funciona

Ao usar `@Web`, o Cursor envia uma consulta para a web, retornando resultados que são então usados como contexto para a IA. Isso permite que a IA forneça respostas mais informadas e precisas, especialmente quando se trata de informações que mudam rapidamente ou que são específicas de um domínio.

### Exemplos de Uso

- **Atualizações de Tecnologia**: Use `@Web` para buscar as últimas atualizações sobre uma tecnologia ou ferramenta que você está usando, garantindo que você esteja sempre atualizado.
- **Resolução de Problemas**: Inclua resultados da web ao tentar resolver um problema complexo, permitindo que a IA considere soluções ou abordagens que foram discutidas online.
- **Pesquisa de Mercado**: Ao desenvolver um produto, use `@Web` para buscar informações sobre concorrentes ou tendências de mercado.

### Sempre Ativo

Você pode fazer o Cursor buscar na web para o chat em cada consulta ativando "Sempre buscar na web" em `Configurações do Cursor` > `Recursos` > `Chat`. Isso é equivalente a usar `@web` em cada consulta.
