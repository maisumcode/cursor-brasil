### `@Code`

O recurso `@Code` permite que você referencie seções específicas de código dentro do Cursor. Isso é especialmente útil quando você precisa fornecer contexto detalhado sobre um trecho de código específico para a IA.

<Frame>
  <img src="/images/context/@code.png" />
</Frame>

## Como Usar

Para usar `@Code`, basta digitar `@` seguido pelo nome do arquivo e a linha ou bloco de código que você deseja referenciar. O Cursor mostrará uma pré-visualização do conteúdo do código para garantir que você está selecionando o trecho correto.
