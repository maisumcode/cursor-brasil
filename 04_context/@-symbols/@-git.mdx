<Info>Atualmente, `@Git` é suportado apenas no Cursor Chat.</Info>

### `@Git`

No Chat do Cursor, você pode usar `@Git` para adicionar commits, diffs ou pull requests do git ao seu prompt.

<Frame>
  <img src="/images/context/@git.png" />
</Frame>

### Casos de Uso Comuns

Um caso de uso comum para `@Git` é permitir que a IA do Cursor examine o diff e procure por bugs ou problemas que possam ser causados pelo diff.

<Frame>
  <img src="/images/context/@git-usecase1.png" />
</Frame>

Você também pode usar `@Diff of Working State` para gerar uma mensagem de commit a partir dos seus diffs atuais.

<Frame>
  <img src="/images/context/@git-commit-message.png" />
</Frame>
