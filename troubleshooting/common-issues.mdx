### Problemas Comuns e Soluções

Esta seção fornece soluções para problemas comuns que você pode encontrar ao usar o Cursor. Se você não encontrar a solução para o seu problema aqui, considere visitar nosso [fórum de suporte](https://forum.cursor.com/) para mais assistência.

#### Atualizações do Cursor

**Problema**: Vejo uma atualização no changelog, mas o Cursor não atualiza.

**Solução**:

- As atualizações são liberadas em etapas. Se a atualização for muito recente, pode ainda não estar disponível para você. Normalmente, leva cerca de 5 horas para alcançar todos os usuários.
- Se você não quiser esperar, baixe o Cursor novamente em nossa [página inicial](https://cursor.com/). Suas configurações e extensões serão mantidas.

#### Login no GitHub

**Problema**: Tenho problemas com meu login no GitHub no Cursor.

**Solução**:

- Use o comando `Sign Out of GitHub` no painel de comandos `Ctrl/⌘ + Shift + P` e tente fazer login novamente.

#### GitHub Codespaces

**Problema**: Não consigo usar o GitHub Codespaces.

**Solução**:

- Atualmente, o Cursor não oferece suporte ao GitHub Codespaces. Considere usar um ambiente de desenvolvimento local.

#### Conexão SSH Remota

**Problema**: Tenho erros ao conectar ao SSH Remoto.

**Solução**:

- O suporte para SSH em máquinas Windows não está disponível. Se você não estiver usando Windows, reporte o problema no [fórum](https://forum.cursor.com/).

#### Proxy Corporativo

**Problema**: Cursor Tab e Cmd K não funcionam atrás do meu proxy corporativo.

**Solução**:

- Defina `"cursor.general.disableHttp2": true` nas configurações (`Cmd/Ctrl + ,` e depois procurar por `http2`) para desativar o HTTP/2, que pode ser bloqueado por alguns proxies.

#### Assinatura Pro

**Problema**: Acabei de me inscrever no Pro, mas ainda estou no plano gratuito no aplicativo.

**Solução**:

- Tente sair e entrar novamente em [Configurações](/get-started/migrate-from-vscode#cursor-settings).

#### Uso Mensal

**Problema**: Quando meu uso será reiniciado novamente?

**Solução**:

- Usuários Pro podem verificar a data de renovação no [Painel](https://cursor.com/settings). Usuários gratuitos podem verificar a data do primeiro e-mail recebido do Cursor.

#### Desinstalação do Cursor

**Problema**: Como desinstalo o Cursor?

**Solução**:

- Siga [este guia](https://code.visualstudio.com/docs/setup/uninstall) substituindo "VS Code" por "Cursor".

### Dicas Gerais para Solução de Problemas

- **Verifique as Configurações**: Certifique-se de que suas configurações estão corretas e atualizadas.
- **Consulte a Documentação**: Use os recursos de documentação do Cursor para entender melhor as funcionalidades e possíveis soluções.
- **Atualize Regularmente**: Mantenha o Cursor atualizado para garantir que você tenha as últimas correções de bugs e melhorias.
